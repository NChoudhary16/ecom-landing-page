How It Works

Smart Detection

Each IoT-enabled dustbin uses sensors to monitor fill levels. When the bin is nearly full, it sends a signal immediately.

Alert Notification

Alerts are pushed in real-time to the municipal dashboard. Authorities instantly see which bins need to be emptied, with notifications displayed visually and on screen.

Charging System

Smart bins use solar or battery power. Charging status and battery health are tracked and shown on the admin dashboard.

Centralized Management

All alerts and bin data appear in the dashboard for the municipality, enabling efficient route planning and maintenance.
